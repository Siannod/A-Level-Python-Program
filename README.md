A python game I made using PyGame for my A Level final project.

This program utilises a variety of algorithms to create random mazes and solve the mazes.

MAZE CREATION
The mazes can be generated using either Prims or Depth First, these two demonstrate a large difference in the way the final maze appears. The player can then play the level while being timed to get on the highscore table.
